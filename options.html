Language

Language of the settings page and side panel UI.

Scope

One entry per line. Use a domain alone for an entire site, or append a path prefix after the domain to match only that section. Subdomains match automatically; * wildcards work in domains and paths.

Keyboard shortcut and toolbar icon can still open the Side Panel on a tab even when Scope would disable link previews — that is intentional so you can explicitly mirror the current page.
Tip: For tighter control, switch to whitelist mode above — the Side Panel only runs on sites you list, so it won't open where you don't need it.
Examples
  • example.com — whole site (every path on that domain)
  • example.com/admin — only /admin and sub-paths (e.g. /admin/users)
  • news.example.com/blog — blog section on one subdomain
  • *shop* — any host name containing shop
Path rules ignore query strings and # fragments. /admin does not match /administrator.

Link scope

By default, eligible outgoing links — including same-tab links and links that open in a new tab — open in the side panel. Uncheck the option below to intercept only new-tab links and leave same-tab navigation to the browser.

Want a link to skip the Side Panel? Hold ⌘ / Ctrl for a new tab, ⇧ for a new window, or ⌥ Option / Alt to open in the current tab. To always skip a page or whole site, add its domain or path prefix under Scope above.

How links open in the Side Panel

Left click opens eligible links in the Side Panel. Enable Also preview on hover below to add hover previews without changing click behavior.
Only middle-click (scroll-wheel click) on a link opens the Side Panel. Left click keeps the browser's normal behavior.
