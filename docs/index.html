How it works

  1. Install from the Chrome Web Store (or load unpacked from GitHub).
  2. Click any target="_blank" link on any site — the page loads in the right-hand Side Panel.
  3. Click the pin icon at the top-right of the Side Panel so it stays open while you switch tabs.
  4. Open Settings to switch trigger mode, language, scope, and more.

Questions

Is it free? Forever?

Yes. The current feature set is permanently free, MIT-licensed, and ad-free. If a paid Pro tier is ever added, all current features stay free.

Does it work on Microsoft Edge?

Yes. Edge 119+ ships the same Side Panel API; the extension installs and runs the same way.

What data does the extension collect?

None. No analytics, no telemetry, no tracking. Settings stay in your Chrome profile via storage.sync. See the Privacy Policy.

Why does some site fail to load in the panel?

A few sites enforce strict frame-ancestors or detect iframe embedding via JavaScript. When that happens you'll see a fallback card with an "Open in new tab" button — one click, you're on the original tab.

Is it affiliated with Arc or Edge?

No. "Arc" and "Edge" are referenced descriptively to explain the split-view interaction model. Side Link Preview is not affiliated with The Browser Company or Microsoft.
